# SIGLStudio v1.0 launch summary

SIGLStudio v1.0 is a polish-and-stability release for the local-first symbolic workstation.

## Included in v1.0

- Unified route intros and onboarding copy across primary workflows.
- Improved command/shortcut discoverability and next-step guidance.
- Hardened validation for publishing bundles, automation packs, and pipeline blueprints.
- Refined import-version checks with explicit compatibility warnings.
- Finalized desktop/browser mode guidance and install/run notes.
- Public-facing release docs, checklist, and known limitations.

## Architecture boundary reaffirmed

- Vibe remains canonical for parsing/lowering/verification/proof truth.
- SIGLStudio remains the product/workstation layer for review, visualization, publishing, and local deterministic handoff.
