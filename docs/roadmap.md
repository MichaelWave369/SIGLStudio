# SIGLStudio Roadmap

## v0.4 (current)
- Presentation mode with clean/focus controls and keyboard navigation.
- Guided sessions with checkpoints, step orchestration, and route launch actions.
- Artifact templates (poster/card) with SVG + JSON exports.
- Provenance/trace alignment components reused across analysis routes.

## v0.5 (next)
- Richer visual themes for presentation/artifacts.
- Symbolic boards/canvas mode for freeform layout.
- Project review pack generation.
- Deeper Vibe-native provenance and trace ingestion.
- Optional collaborative handoff formats (file-based only).
